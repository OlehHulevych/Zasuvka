Třída Product

Slouží jako Model pro správu produktů (inzerátů). Data jsou ukládána do JSON souboru (data/Products.json). Třída řeší CRUD operace (Vytvoření, Čtení, Úprava, Mazání), filtrování, stránkování a správu obrázků spojených s produkty.
