Třída FavoriteList

Slouží k manipulaci se seznamy oblíbených položek (Wishlists), které jsou uloženy v lokálním souboru JSON (data/FavoriteLists.json). Funguje jako náhrada databáze pro ukládání vztahu Uživatel <-> Seznam oblíbených.
