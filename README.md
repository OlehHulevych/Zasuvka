# Produktová dokumentace: Zasuvka 
## Přehled produktu (Product Overview)
**Vize:** Zasuvka - to web-stranka kde uživatel může koupit použite zbožit za určitou cenu a takže prodavate své zbožit a vydělat na tom.

**Hlavní problém**: Zde uživatel může koupit levnějši zboži a takže prodat které nepotřebuje 

**Cílová skupina**: uživatelé, Firmy, internet-obchody

## Funkční požadavky (Functional Requirements)

**Správa profilu**: Registrace, přihlášení (email).

**Hlavní funkcionalita**:může přidavat, editovat nebo mazat své inzeraty

**Vyhledávání a filtrace**: Možnost najít položky podle kategorií a nebo podlé určitého slova.

## Uživatelské role (User Roles)
**Admin**: Má plný přístup ke správě uživatelů a databáze.

**Běžný uživatel**: Může prohlížet a editovat pouze svůj obsah.

**Host**: Může pouze prohlížet veřejné části webu a prohližet inzeraty.

## Technická stranka
**Frontend:** HTML, CSS, JS.

**Backend:** PHP, JSON

**Responzivita**: Aplikace musí fungovat na mobilu, tabletu i desktopu.

**Bezpečnost**: Šifrování dat.
